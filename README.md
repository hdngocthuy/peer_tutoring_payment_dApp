# 🚀 Stellar Tutor Payment Contract

## 📌 Description

This project implements a decentralized tutor payment tracking system using Soroban smart contracts on the Stellar network.

The goal is to provide a transparent and immutable way for students to record payments made to tutors. Instead of relying on centralized systems, all transactions are stored on-chain, ensuring trust, auditability, and security.

This project is designed for learning and demonstrating how blockchain technology can be applied to real-world education and freelance payment scenarios.

---

## ✨ Features

* 🔐 **Authentication**

  * Ensures only the student can authorize a payment (`require_auth`)

* 💰 **Payment Recording**

  * Records each payment with student, tutor, amount, and timestamp

* 📊 **Total Payment Tracking**

  * Stores cumulative payment between each student–tutor pair

* 🧾 **Payment History**

  * Maintains a full on-chain history of all transactions

* ⚡ **Lightweight & Efficient**

  * Uses Soroban storage for optimized on-chain data handling

## 🚀 Project Vision

This project aims to build a decentralized payment system for tutor–student interactions.

In the future, it can evolve into a full dApp that:

* Supports real token transfers
* Provides a user-friendly web interface (React)
* Enables filtering and analytics on payment history
* Integrates decentralized identity (DID)

The long-term vision is to create a trustless and transparent financial system for education and freelance services.
